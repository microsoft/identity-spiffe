# Support

This repository is a prototype and sample implementation. It is provided as-is
with no production support SLA.

## Questions and usage help

Use GitHub issues or discussions, when enabled, for:

- Setup and deployment questions
- Documentation gaps
- Reproducible bugs in the sample
- Feature requests or scenario suggestions

Include the deployment mode, operating system, relevant command output, and the
smallest reproduction you can provide. Do not include credentials, tenant IDs,
subscription IDs, tokens, secrets, or production data.

## Security vulnerabilities

Do not report security vulnerabilities through public GitHub issues. Follow the
instructions in [`SECURITY.md`](SECURITY.md).
