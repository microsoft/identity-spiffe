# Code of Conduct

This project has adopted the
[Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/).

Participants are expected to follow the code of conduct in all project spaces,
including issues, pull requests, discussions, and community channels associated
with the project.

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported through the contact options listed on the Microsoft Open Source Code
of Conduct page.
